## Technology Stack

### Frontend
- **Framework**: React 18
- **UI Library**: Material-UI (MUI)
- **State Management**: Redux
- **Real-time**: Socket.IO Client
- **Web3**: Ethers.js, Web3.js
- **Build Tool**: Webpack (via react-scripts)

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB + Mongoose
- **Authentication**: JWT
- **Real-time**: Socket.IO
- **Blockchain**: Web3.js, Ethers.js, TronWeb

### Infrastructure
- **Database**: MongoDB 
- **Process Manager**: Concurrently (development)
- **API Integration**: Tatum, Infura
